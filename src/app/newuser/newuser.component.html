Create A New User

Quick Crypt uses passkeys, along with your own per message password, to encrypt message and help ensure that an untrusted website cannot easily trick you into exposing unencrypted data. Quick Crypt associates your passkey with a user name, which must be 6 to 31 character long. The user name should help you remember what it is used for. An email address acceptable, but not required. If needed, you can create multiple user accounts each with one or more passkeys.

When you click the Create new user and passkey button below, you should be promoted to create or update a passkey on your system. You should choose to create a new passwkey. You can create a passkey using physical FIDO2 device, a mobile device, a browser based password manager, or a dedicated password manager. We find that dedicated password managers like Bitwarden or 1Password are the easiest to use, work consistently across devices, and allow for backup.
