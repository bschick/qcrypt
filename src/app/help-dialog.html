Quick Crypt Help

Quick Crypt encrypts and decrypts text using a password you provide. Quick Crypt uses cryptographic features built into modern browsers, ensuring that your password, clear text, and even encrypted cipher text do not unexpectedly traverse the internet. A typical use is to encrypt text before storing or sending it through an untrusted channel. Your original text can only be retrieved by someone who knows your password. Remember your password, or you won't be able to retrieve the original text either.

To use Quick Crypt, paste or upload the text you want to be encrypted in the field on the left labeled Enter Clear Text, then click the Encrypt button and enter a password. You can enter an optional password hint to remind yourself which password you used. The hint is stored in the clear with the encrypted text, so don't make it too revealing. The encrypted results are displayed to the right and can be copied or saved to disk. To decrypt information previously encrypted with Quick Crypt, paste or open the encrypted text in the field on the right labeled Enter Cipher Text and click the Decrypt button. If you've forgotten your password, you are out of luck unless you have access to a supercomputer and time to spare.

You can customize some of Quick Crypt's functionality by opening the Advanced Encryptions Options panel and making changes. Other than your password choice, Quick Crypt does not provide options known to be insecure, so adjusting settings should be safe. If you want even more security, you can encrypt something multiple times with the Loop Encrypt option; just be sure to use a different password for each loop.

Technical Details
  • Cryto Library: Browser native Web Crypto API
  • Key Derivation: PBKDF2 using SHA-512 and selectable iterations (default 0.5s of hashing or 800K)
  • Key Length: 256 bits (max for web crypto)
  • Encryption: AES with selectable mode of operation and optional HMAC-SHA256
  • Random Data: random.org or window.crypto.getRandomValues
